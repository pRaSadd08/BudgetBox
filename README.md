### How to Install
Git Clone
git clone https://github.com/pRaSadd08/BudgetBox

Backend setting
cd backend

pip install -r requirements.txt

python manage.py makemigrations

python manage.py migrate

python manage.py runserver

#### Open http://127.0.0.1:8000/posts/
